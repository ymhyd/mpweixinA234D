### 五、项目介绍
项目编号：mpweixinA234D

基于微信小程序的零食商城销售系统，方便用户随时随地浏览选购各类零食商品，提升商家运营管理效率。

角色：管理员、用户、商家

用户功能：注册登录、商品浏览、搜索查询、加入购物车、在线下单、订单管理、收藏商品、评价晒单、个人中心。

商家功能：登录、商品管理、订单管理、活动促销、库存管理、店铺设置。

管理员功能：登录、用户管理、商家管理、商品信息管理、订单管理、分类管理、活动促销管理、系统日志。

数据库表：18张
